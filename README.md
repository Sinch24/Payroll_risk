# Payroll Risk Scoring

This project identifies high risk payroll records using employee attendance and salary data.

## Dataset
Synthetic payroll dataset created for learning purposes.

## Steps
- Data loading
- Exploratory Data Analysis
- Model training
- Model comparison
- Feature importance

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Result
Random Forest gave the best performance based on F1 score.
